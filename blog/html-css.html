Inline vs Inline Blocks

The main difference between inline and inline block is that an inline block allows you to;

  1. set width
  2. set hight

and with in-line the top and bottom margins as well as the paddings are not respected.

A block-line ellement always starts on a new line and the browser always adds space before and after the element.

Any element within a program, document or message is called inline or in-line,

but does not include any code loaded from an external file.

In any web page, almost all the contents are inline except the CSS and image files.

Any element that starts a new line for a paragraph and uses the full page or container is called block elements.

It has one line or multiple lines and possesses a line break before and after the elements in the line.

CSS can be used for formatting the block element
